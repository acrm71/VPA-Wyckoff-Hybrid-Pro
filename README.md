## Overview

VPA Wyckoff Hybrid Pro Professional is a modular TradingView indicator designed to combine institutional trading concepts into one integrated decision-support system.

The project combines:

- Volume Price Analysis (VPA)
- Wyckoff Methodology
- Market Structure
- Relative Volume
- Anchored VWAP
- Fair Value Gaps
- Supply & Demand
- Institutional Accumulation / Distribution
- Multi-Timeframe Analysis
- AI-style Confluence Scoring

The objective is **not** to generate buy/sell signals.

Instead, the indicator evaluates multiple independent factors and presents a weighted confidence score to assist discretionary decision making.
